Power BI Credit Card Transaction Dashboard

📁 Project Description
This project analyzes credit card transaction data using Power BI. It includes Cards like revenue, interest, amount and Count of transaction over the period of time.

🧩 Features
- Dynamic filters
- Revenue-wise transaction breakdown
- Quaterly Transaction trends (line  & stacked column chart)
- Card wise Revenue details (Table)

🗂 Files
- `Credit Card Report.pbix` – Main report file
- `credit_card/` – Source data (CSV/Excel)
- `screenshot-Credit Card Transaction Report/` – Dashboard preview images
